清除论坛：F12+控制台+localStorage.clear（） 
python服务器开启：python -m http.server 自定义端口（也可以不自定义，不自定义就是8000端口） 
python服务器地址：http://localhost:8000（或是你的自定义端口） 
python服务器快捷开启脚本： 
---@echo off 
---python -m http.server 
---pause 
把“---”去掉，建立一个“python-server-run.bat”文件 以记事本打开后把文本粘贴进去 然后保存，
双击出现类似“Serving HTTP on ：： port 8000 （http://[：：]：8000/） ...”的内容就成功了
